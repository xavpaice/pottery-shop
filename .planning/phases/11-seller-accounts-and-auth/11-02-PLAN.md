## Tasks

### Task 1: Extend SessionData with SellerID
Read `internal/middleware/session.go` and add `SellerID int64 \`json:"seller_id,omitempty"\`` to the `SessionData` struct. Zero value means anonymous/buyer. Existing sessions without the field will deserialize to SellerID=0 — no migration needed.

Files:
- `internal/middleware/session.go` — add `SellerID int64` field

### Task 2: AuthHandler — login, register, logout
Create `internal/handlers/auth.go`:

```go
type AuthHandler struct {
    sellers   *models.SellerStore
    sessions  *middleware.SessionManager
    templates *template.Template
}
```

Methods:
- `ShowLogin(w, r)` — GET /login; renders login.html with optional flash from session
- `Login(w, r)` — POST /login; validates form, calls GetByEmail + CheckPassword; if `!seller.IsActive` renders login.html with "Your account is pending approval" message; on success sets session.SellerID and redirects to /dashboard
- `ShowRegister(w, r)` — GET /register; renders register.html
- `Register(w, r)` — POST /register; validates email/password/name fields (server-side only), calls SellerStore.Create, sets flash "Registration submitted — await admin approval", redirects to /login
- `Logout(w, r)` — POST /logout; clears session.SellerID (set to 0), redirects to /

Also add `DashboardHandler` struct (or receiver on AuthHandler) with:
- `requireSeller(next http.HandlerFunc) http.HandlerFunc` — middleware guard; reads session; if SellerID==0 redirect /login; if seller is_active=false redirect /login with flash
- `Dashboard(w, r)` — GET /dashboard; renders dashboard.html with seller name

Files:
- `internal/handlers/auth.go` — new file

### Task 3: Register routes in main.go
Read `cmd/server/main.go` and add:
- Instantiate `AuthHandler` with SellerStore, SessionManager, templates
- Register routes: GET+POST /login, GET+POST /register, POST /logout, GET /dashboard (guarded)

Files:
- `cmd/server/main.go` — add AuthHandler init and route registration

All templates extend the existing base layout pattern used in the codebase.
