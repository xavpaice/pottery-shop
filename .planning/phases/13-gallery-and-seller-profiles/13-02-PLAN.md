### Task 1: Update Home and Gallery handlers to use seller JOIN query
Read `internal/handlers/public.go`.

Update `PublicHandler.Home()` and `PublicHandler.Gallery()` to call `ProductStore.ListAllWithSeller(ctx)` (added in Phase 12 Plan 1) so that each `Product` in the result carries `SellerID` and `SellerName`.

For optional gallery seller filter: in `Gallery()`, check `r.URL.Query().Get("seller")`; if non-empty and a valid int64, call `ProductStore.ListBySeller(ctx, sellerID)` instead. If the seller ID is invalid, ignore the filter (no error).

Files:
- `internal/handlers/public.go` — update Home() and Gallery() to use ListAllWithSeller; add optional seller filter to Gallery()

### Task 2: Update product card templates
Read `templates/index.html` and `templates/gallery.html` before editing.

In each product card, add seller attribution below the product title or image:
```html
<p class="seller-name">by <a href="/seller/{{.SellerID}}">{{.SellerName}}</a></p>
```

Use a `{{if .SellerName}}` guard so cards without a seller (NULL seller_id rows) render cleanly.

Files:
- `templates/index.html` — add seller name link to product cards
- `templates/gallery.html` — add seller name link to product cards

### Task 3: Update product detail page
Read `templates/product_detail.html` and `internal/handlers/public.go ProductDetail()`.

Ensure `ProductDetail()` uses a query that JOINs sellers (or calls `GetByID` with a JOIN) so `product.SellerName` is populated.

Add to the product detail template, near the title:
```html
{{if .Product.SellerName}}
<p>By <a href="/seller/{{.Product.SellerID}}">{{.Product.SellerName}}</a></p>
{{end}}
```

Files:
- `templates/product_detail.html` — add seller attribution block
- `internal/handlers/product.go` or `internal/handlers/public.go` — ensure ProductDetail uses a seller JOIN (read the file first to find the relevant handler)
