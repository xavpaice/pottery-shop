## Tasks

### Task 1: ReadingsAPI handler method
Read `internal/handlers/firing_log.go`.

Add `ReadingsAPI(w http.ResponseWriter, r *http.Request)` to `FiringLogHandler`:

1. Read session; if `session.SellerID == 0` → write `401 Unauthorized` JSON error and return
2. Parse `{id}` from URL path
3. Call `h.logs.GetReadingsForAPI(ctx, id, session.SellerID)` — returns nil if not found or wrong owner
4. If nil → write `403 Forbidden` JSON error and return
5. Build response struct:
   ```go
   type readingsResponse struct {
       Readings []readingItem `json:"readings"`
   }
   type readingItem struct {
       ElapsedMinutes int     `json:"elapsed_minutes"`
       Temperature    float64 `json:"temperature"`
       GasSetting     string  `json:"gas_setting"`
       FlueSetting    string  `json:"flue_setting"`
   }
   ```
6. Set `Content-Type: application/json`, write `json.Marshal(response)` to response body
