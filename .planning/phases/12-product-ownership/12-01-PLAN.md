## Tasks

### Task 1: Goose migration — add seller_id to products
Create `db/migrations/00003_add_product_ownership.sql`:

```sql
-- +goose Up
ALTER TABLE products ADD COLUMN seller_id BIGINT REFERENCES sellers(id);

-- Assign existing products to the first admin seller
UPDATE products
SET seller_id = (SELECT id FROM sellers WHERE is_admin = true ORDER BY id LIMIT 1)
WHERE seller_id IS NULL;

-- +goose Down
ALTER TABLE products DROP COLUMN seller_id;
```

The UPDATE is a best-effort data migration. If no admin seller exists yet (e.g., a fresh schema from scratch with both migrations applied before bootstrap), seller_id remains NULL. The bootstrap in Plan 11-03 runs after migrations, so on a real deployment the admin seller will exist.

Files:
- `db/migrations/00003_add_product_ownership.sql` — new file
