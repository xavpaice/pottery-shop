# clay.nz brand kit

This package is optimized for web use with a flatter, cleaner logo treatment.

## Included assets
- `logo-primary-stacked-light.svg`
- `logo-primary-stacked-dark.svg`
- `logo-horizontal-light.svg`
- `logo-horizontal-dark.svg`
- `logo-mark-light.svg`
- `logo-mark-dark.svg`
- `favicon.svg`
- `favicon.ico`
- `favicon-16.png`
- `favicon-32.png`
- `favicon-48.png`
- `apple-touch-icon.png`
- `brand.css`
- `index-example.html`

## Recommended usage
- Navbar: `logo-horizontal-light.svg`
- Footer / hero: `logo-primary-stacked-light.svg`
- Social avatar / compact UI: `logo-mark-light.svg`
- Dark background sections: use the `-dark.svg` variants
- Browser tab favicon: `favicon.ico` plus `favicon.svg`

## Color palette
- Clay: #B56E4A
- Clay dark: #8C5237
- Charcoal: #1F1A17
- Stone: #6F655D
- Bone: #F5F1EB
- Sand: #E7DED3

## Typeface guidance
- Display / headings: Baskerville / Iowan Old Style / Georgia-style serif
- Body / UI: Inter or system sans-serif

## Notes
These files are vector-first. The symbol is symmetrical left-to-right, with asymmetry only in the internal cut.
